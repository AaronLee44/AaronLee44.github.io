计算大数阶乘的思路

由于计算阶乘的过程中，主要问题是——数字不断相乘之后由于结果剧增使普通的C++整型数据类型无法存储，所以我们可以定义一个int类型的数组来存储计算的结果，数组的每个单元都只保存一位，例如arr[0]只保存个位，以此类推。这样，我们在计算$1_2_…*n$过程中，先将arr[0]设置为1，其他元素均为0，然后用2乘以整个数组，考虑每一位上的进位和余数，然后更新这个arr[i]单元，最后迭代即可。

如下所示，开始时设置最低位arr[0]=1，下面为$n=1$时的结果：

arr[N-1] … … … … arr[2] arr[1] arr[0]
0 0 0 0 0 0 0 1

当$n=2$时，用$2$乘以上面整个数组，得到：

arr[N-1] … … … … arr[2] arr[1] arr[0]
0 0 0 0 0 0 0 2

当$n=3$时，用$3$乘以上面整个数组，得到：

arr[N-1] … … … … arr[2] arr[1] arr[0]
0 0 0 0 0 0 0 6

当$n=4$时，用$4$乘以上面整个数组，数组更新为：

arr[N-1] … … … … arr[2] arr[1] arr[0]
0 0 0 0 0 0 2 4

…

以此类推，在此过程中值需要考虑每一位上的进位和进位之后的余数。最后数组中每个单元的数字就是阶乘结果的每一位。
